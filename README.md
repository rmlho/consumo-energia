# ⚡ Consumo de Energia

Sistema de terminal para cadastrar equipamentos domésticos, registrar o uso diário e calcular o consumo de energia elétrica (kWh) e o custo na fatura.

Desenvolvido como projeto da disciplina de **Algoritmos** — UEPB, 2025.1.

---

## 🖥️ Demo

[![Assistir demo](https://img.youtube.com/vi/_gwH-lDgb1M/0.jpg)](https://youtu.be/_gwH-lDgb1M)

---

## 💡 Funcionalidades

- Cadastro de equipamentos com nome e potência (W)
- Registro de horas de uso por dia
- Cálculo automático de consumo em kWh
- Estimativa de custo com base na tarifa informada
- Listagem de todos os equipamentos cadastrados

---

## 🗂️ Estrutura do projeto
